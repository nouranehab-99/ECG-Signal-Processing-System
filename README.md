# ECG Signal Processing & Filtering System

DSP-based ECG filtering and analysis project developed using Python.

## Features
- FIR and IIR ECG filtering
- FFT frequency-domain analysis
- R-peak detection
- HRV extraction
- Zero-phase filtering

## Tools & Technologies
- Python
- NumPy
- SciPy
- Matplotlib

## Results
- Improved ECG signal quality using digital filtering
- Achieved accurate heartbeat analysis and HRV metrics
